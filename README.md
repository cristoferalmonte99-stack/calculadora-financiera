# 📊 Calculadora Financiera

## 🧠 ¿Qué hace este programa?

Esta aplicación permite resolver ejercicios de **matemática financiera**, incluyendo cálculos de:

* Interés simple
* Interés compuesto (en desarrollo)
* Capital, monto, tasa y tiempo

El objetivo es facilitar la práctica y comprensión de fórmulas financieras mediante un programa interactivo en Python.

---

## ⚙️ Tecnologías utilizadas

* Python

---

## 📁 Estructura del proyecto

```
CALCULADORAFINANCIERA/
│
├── logica_interes_simple.py
├── logica_interes_compuesto.py
├── menu_interes_simple.py
├── main.py
```

---

## ▶️ ¿Cómo usar el programa?

1. Ejecutar el archivo `main.py`
2. Seleccionar una opción del menú
3. Ingresar los datos solicitados
4. Obtener el resultado del cálculo
